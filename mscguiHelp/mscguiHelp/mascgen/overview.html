MASCGEN

The MOSFIRE Automatic Slit Configuration Generator (MASCGEN) is a tool used to generate slit configurations based on a prioritized list of astronomical targets and alignment stars, given specified mask parameters and search criteria. Using the inputs entered into the Inputs panel (or from a saved parameter file), MASCGEN will dither the pointing over the search parameters and for each pointing, determine the best mask that can be created. "Best" in this sense is the mask with the highest total priority score, which is the sum of the priorities of the objects that are in the mask. It then keeps track of the pointing that provides the overall highest priority score, and presets this to the user as the optimized mask and newly formed mask configuration.

Help for MASCGEN is divided into Target Lists, Inputs, Outputs, and Running MASCGEN.
